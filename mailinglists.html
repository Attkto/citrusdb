Announcements

This announcements list is for announcements of new versions of citrus, security, and critical updates. It is not a discussion list. You can join the list by going here: https://lists.sourceforge.net/lists/listinfo/citrusdb-announce

Users

This is a discussion list for users of CitrusDB. Ask questions or get help with usage, installation, setup. You can join the list by going here: https://lists.sourceforge.net/lists/listinfo/citrusdb-users
