Help other Users

Sign up for the citrusdb-users mailing list and help answer questions of other users.

Source Code

Please read the coding standards. You can send patches to branches on launchpad or by email to paul at citrusdb.org. You can check out the latest development branch of citrusdb using the bzr version control program from the recent branches of citrusdb.

Translations

You can help by updating the translations included with the release or contribute new translation files. The us-english file is always the most up-to-date. Please use that to begin any new translations. Email them to: paul at citrusdb.org.

Let the world know about CitrusDB

Write about this project or link to us on your website or blog. Comment on or rate this project at freshmeat.net. Vote for or review this project at Hotscripts.com.
