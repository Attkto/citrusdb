Development

CitrusDB is developed under the GPL version 2 license. This project was started by Paul Yasi, originally to fulfill his own customer database needs.

Do you want to contribute source code? Please read the coding standards You may send code to paul at citrusdb.org or post it to a branch on launchpad

Translations are also needed, you can help by updating the translations included with the release or contribute new translation files. Please email them to: paul at citrusdb.org.

Bug reports and fixes are welcome. You can look through the list of bugs in the Bug Tracking page. It would be helpful to sign up for a new account in the bug tracker first so you can be notified of changes to the status of your bug.
